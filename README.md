# SARUCREW Claude Code スタートアップガイド

このガイドに沿って進めれば、**15分でClaude Codeが使える状態**になります。
ITに詳しくなくても大丈夫です。一つずつ進めていきましょう。

---

## Step 1: VSCode をインストール（まだの人のみ）

VSCode（Visual Studio Code）は無料のエディタです。今回はClaude Codeの「入れ物」として使います。

1. 以下のリンクを開く
   - https://code.visualstudio.com/
2. 「Download for Windows」をクリック
3. ダウンロードしたファイルを開いてインストール（全部「次へ」でOK）

[スクショ: VSCode ダウンロードページ]

> 既にVSCodeが入っている人はこのステップをスキップしてください。

---

## Step 2: Claude Code 拡張をインストール

1. VSCodeを開く
2. 左側の四角いアイコン（拡張機能）をクリック
3. 検索バーに「Claude Code」と入力
4. 「Claude Code」（Anthropic公式）の「インストール」をクリック

[スクショ: VSCode拡張機能の検索画面 → Claude Code]

---

## Step 3: Team plan アカウントでログイン

1. 管理者から届く **招待メール** のリンクをクリック
2. Anthropicアカウントを作成（またはログイン）
3. VSCodeに戻ると、Claude Codeが自動で認証されます

[スクショ: Team plan 招待メール → ログイン画面]

> Team planを使うことで、入力したデータがAIの学習に使われません。
> 必ず個人アカウントではなく、**会社のTeam planアカウント**でログインしてください。

---

## Step 4: SARUCREW共通設定をインストール

ここが一番大事なステップです。会社のセキュリティルールと安全設定を自動で入れます。

1. VSCodeの上部メニューから「ターミナル」→「新しいターミナル」を選択
   （またはキーボードで `Ctrl + `` ）

[スクショ: ターミナルの開き方]

2. 以下のコマンドを **まるごとコピーして** ターミナルに貼り付け、Enterを押す

```bash
curl -sL https://raw.githubusercontent.com/kokihasegawa0624/claude-setup/main/setup.sh | bash
```

3. 「セットアップ完了！」と表示されればOK

[スクショ: セットアップ完了の画面]

> このコマンドは以下の2つを自動で設定します：
> - **CLAUDE.md** — AIの行動ルール（セキュリティポリシー、日本語対応等）
> - **settings.json** — 危険な操作のブロック（ファイル削除の禁止等）

---

## Step 5: 動作確認

1. ターミナルに `claude` と打ってEnter

```
claude
```

2. Claude Codeが起動したら、試しに何か聞いてみましょう：

```
SARUCREWという会社の社内アシスタントとして自己紹介してください
```

3. 日本語で返答が来ればセットアップ完了です！

[スクショ: Claude Code起動 → 返答画面]

## 基本の使い方

### 毎日やること

```
① VSCodeを開く
② ターミナルを開く（Ctrl + `）
③ 「claude」と打ってEnter
④ あとは日本語でチャットするだけ
```

### よく使うコマンド

| コマンド | 何をするか |
|---------|-----------|
| `/clear` | 会話をリセットして最初から始める |
| `/compact` | 長い会話を要約してスッキリさせる |
| `@ファイル名` | 特定のファイルをClaudeに渡す |
| `/help` | ヘルプを表示する |

### すぐ試せること

- 「この業界の最新トレンドを3つにまとめて」
- 「このメールを英語に翻訳して、丁寧なビジネストーンにして」
- 「この議事録からネクストアクションを抜き出して」
- 「AとBのメリデメを表にまとめて」

> **ポイント**: 曖昧に聞くより具体的に指示する方が良い結果が出ます。
> 「いい感じにして」より「箇条書き3点、各50字以内で」のように伝えましょう。

## やっていいこと・やってはいけないこと

### やっていいこと ✅
- 公開情報の調査・分析
- 自分の業務文書の要約・翻訳・添削
- アイデア出し・ブレスト
- 社内データの分析（Team環境内で）

### やってはいけないこと ❌
- 顧客の個人情報（氏名・連絡先・契約内容等）を入力する
- パスワード・APIキー・社内認証情報を入力する
- Team環境以外のAIツール（無料版ChatGPT等）に業務データを入れる
- AIの出力をファクトチェックせずにそのまま社外に出す

## Desktop / Web版を使いたい場合

ターミナルが苦手な人は、ブラウザやアプリで使える **Desktop / Web版（claude.ai）** もあります。
同じTeam planアカウントでログインすれば、データ保護は同じです。

1. ブラウザで [claude.ai](https://claude.ai) にアクセス
2. Team planアカウントでログイン
3. **チャット開始時に、以下のルールファイルを貼り付けてから作業してください**

👉 [SARUCREW_claude_desktop_rules.md](./SARUCREW_claude_desktop_rules.md)

> **注意**: Desktop / Web版にはセキュリティ設定の自動適用がありません。
> 毎回ルールを読み込ませないと、個人情報の警告やファクトチェックのリマインドが効きません。

## 困ったときは

| 状況 | 対処法 |
|------|--------|
| Claude Codeが起動しない | VSCodeを再起動してもう一度 `claude` と入力 |
| ログインできない | 招待メールのリンクからやり直す。管理者に問い合わせ |
| 変な動きをする | `/clear` で会話をリセット |
| わからないことがある | 各部署のAIリーダーに相談 |
